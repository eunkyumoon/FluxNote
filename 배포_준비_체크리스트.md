# 배포 준비 체크리스트

## ✅ 완료해야 할 항목

### 1. GitHub 계정
- [ ] GitHub 계정이 있음
- [ ] GitHub에 로그인 가능

### 2. GitHub 저장소 생성
- [ ] 새 저장소 생성 완료
- [ ] 저장소 이름: `fluxnote` (또는 다른 이름)
- [ ] 저장소 URL 복사해두기 (예: `https://github.com/사용자명/fluxnote.git`)

### 3. 준비 완료
- [ ] 위 항목 모두 완료
- [ ] 저장소 URL 준비됨

---

## 다음 단계

위 항목을 모두 완료하면, 다음 명령어들을 실행하면 됩니다:

```bash
git init
git add .
git commit -m "첫 번째 커밋: FluxNote 메모 앱"
git branch -M main
git remote add origin https://github.com/사용자명/fluxnote.git
git push -u origin main
```

**⚠️ 주의**: `사용자명`을 여러분의 GitHub 사용자명으로 바꿔주세요!
